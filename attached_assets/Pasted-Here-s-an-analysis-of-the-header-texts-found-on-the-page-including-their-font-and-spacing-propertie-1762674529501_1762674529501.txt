Important Notes for Replit:

Font Availability: For BB-CondBold, you'll need to make sure this font is available in your Replit project. This usually means importing it via @font-face rules in your CSS or linking a font service (like Google Fonts) if it's a web font. If not, the browser will fall back to "Helvetica Neue" or the next available font.
Unitless line-height: It's often a best practice to use unitless line-height values (e.g., 1.15 instead of 16.1px) as they scale better with different font sizes. You can calculate this by dividing the line-height by the font-size.
Default Values: word-spacing: 0px and font-variant-ligatures: normal are often default browser styles, so you might not need to explicitly declare them unless you're overriding a different style.
